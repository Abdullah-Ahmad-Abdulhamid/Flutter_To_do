# 📝 Flutter To-Do List App

A simple and clean To-Do List application built with Flutter.  
The app helps users manage their daily tasks efficiently with a modern UI and smooth experience.

---

## ✨ Features

- ➕ Add new tasks with title, date, and time
- ✔️ Mark tasks as completed
- 📦 Archive tasks
- 🗑️ Delete tasks permanently
- 🔄 Switch between All / Done / Archived tasks
- 💾 Local database storage using SQLite
- 📱 Clean and responsive UI

---

## 🧱 Architecture

The app follows a clean structure:

- **Presentation Layer**
  - UI screens and widgets
- **Business Logic Layer**
  - State management using `Cubit (flutter_bloc)`
- **Data Layer**
  - SQLite database handling

---

## 🛠️ Tech Stack

- Flutter
- Dart
- flutter_bloc
- sqflite
- intl
